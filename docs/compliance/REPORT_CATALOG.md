# Report Catalog (Phase 0 Scaffold)

## Planned Report Types
- Significant events and alterations
- Metering snapshots and deltas
- Cashless account statement
- Liability and transfer exception reports

## Required Report Intervals
- DTD
- MTD
- YTD
- LTD

## Standard Metadata for Every Report
- Operator identifier
- Report title
- Selected interval
- Generation timestamp
- "No Activity" marker when applicable
